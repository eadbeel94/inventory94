Tecnologias utilizadas en esta aplicacion

FRONTEND HTML
El lenguaje de marcado clásico con lo que está construida toda la web.
FRONTEND CSS
El lenguaje de estilizado con el que se mejora el aspecto visual de la web.
FRONTEND JAVASCRIPT
Uno de los lenguajes de programación más populares dándole un toque de animación a nuestra web.
BACKEND NODE JS
Javascript siendo el lenguaje utilizado del lado del cliente, ahora también puede correr funciones del lado del servidor.
BACKEND MONGO DB
Base de datos NO-SQL de acceso rápido que utiliza el acomodo de datos en tipo JSON.

Framework y librerias adicionales utilizadas

FRONTEND CSS BOOSTRAP 4
Framework de CSS el cual nos permite estilizar de una manera rápida y sencilla nuestro HTML.
FRONTEND CSS FONTAWESOME 5
Librería de iconos y fuentes las cuales les dará un toque profesional a su WEB.
FRONTEND CSS ANIMATE.CSS
Librería de CSS que nos permite animar nuestros componentes HTML.
BACKEND EXPRESS JS
Framework de Node JS, express nos permite correr funciones de servidor de una manera más sencilla.
BACKEND MULTER
Modulo/plugin utilizado en Node JS el cual nos permite manipular archivos del lado del servidor.
BACKEND PASSPORT JS
Con el apoyo de este plugin se puede manejar de una manera segura el uso de sesiones durante los logins.
